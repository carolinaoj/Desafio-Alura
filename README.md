# 🎁 Amigo Secreto

Una aplicación web sencilla desarrollada con **HTML**, **CSS** y **JavaScript** puro para organizar un sorteo de amigo secreto de forma aleatoria.

---

## Permite:
- Agregar nombres de amigos uno por uno.
- Validar que los nombres no estén vacíos ni contengan números.
- Mostrar visualmente la lista de personas que participan.
- Sortear aleatoriamente un "amigo secreto" con solo un clic.
- Ver el nombre sorteado directamente en la pantalla.
